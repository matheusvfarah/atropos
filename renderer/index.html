Sincronização (Dispositivos)

Para ativar o sync entre dispositivos:

  1. Criar um repositório PRIVADO no GitHub para o vault
  2. No terminal, dentro do vault:
git init
git remote add origin https://github.com/usuario/meu-vault.git
git add -A && git commit -m "initial vault"
git push -u origin main
  1. No segundo dispositivo, clonar o repositório como vault
  2. O Grafo Líquido detecta o remote automaticamente
Repositório PRIVADO é fortemente recomendado — o vault contém notas pessoais e o conteúdo será enviado para o GitHub.
